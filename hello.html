Báo cáo tuần 1

HTML là gì ?

  • HTML là viết tắt của Hyper Text Markup Language
  • HTML là ngôn ngữ đánh dấu tiêu chuẩn để tạo các trang Web
  • HTML mô tả cấu trúc của một trang Web
  • Các phần tử HTML cho trình duyệt biết cách hiển thị nội dung

CSS là gì ?

  • CSS là viết tắt của Cascading Style Sheets
  • CSS mô tả cách các phần tử HTML được hiển thị trên màn hình, giấy hoặc trong các phương tiện khác
  • CSS tiết kiệm rất nhiều công việc. Nó có thể kiểm soát bố cục của nhiều trang web cùng một lúc
  • Các bảng định kiểu bên ngoài được lưu trữ trong các tệp CSS
  • Các thuộc tính cơ bản của css

Hình ảnh trực quan của một cấu trúc web

Một website hoạt động như thế nào?

I.Các thành phần của một web

Web server và web hosting

  • Web server: là một máy tính chứa file và database tạo nên website của bạn.
  • Web server được xác định bởi địa chỉ IP
  • Web hosting là dịch vụ sử dụng máy chủ để lưu trữ các tập tin, tài nguyên
  • Vì nhu cầu để mua máy chủ hiện nay thì chỉ có các doanh nghiệp mới có nhu cầu,còn lại với việc học tập hay cửa hàng nhỏ thì gói hosting là vừa phải không cần phải mua cả server.

Tên miền(domain)

  • Tên miền là tên thay thế cho địa chỉ IP của máy chủ web, bởi địa chỉ IP là một dãy số rất khó nhớ

Dữ liệu (Data)

  1. Database hay cơ sở dữ liệu là những thông tin về người dùng (thường xuất hiện ở web động)
  2. Web data là dữ liệu của web dưới các dạng cơ bản như hình ảnh, âm thanh, văn bản

Mã nguồn (Source Code)

Đó chính là nội dung, chức năng hiển thị lên cho người dùng xem, đọc, truy cập. Tùy vào chức năng và yêu cầu của website, mà mã nguồn có thể nhiều hay ít câu lệnh, phức tạp hoặc đơn giản.

Giao diện người dùng(front-end)

Giao diện người dùng là định dạng trang web được trình bày trên màn hình của máy tính của người xem (máy khách) được xem bằng các phần mềm trình duyệt web như Chrome, Firefox,…

Ngoài máy tính ra còn có các dạng giao diện khác trên các nền tảng khác nhau

II.Các công cụ hỗ trợ việc truy cập vào web

Các trình duyệt web

Nhiệm vụ của trình duyệt web là làm cho nội dung trong các file HTML, XML hiển thị theo bố cục định sẵn trong file CSS. Thông qua nó, người dùng có thể đọc/xem và tương tác với website.

Mạng máy tính

Mạng máy tính hỗ trợ một số lượng lớn các ứng dụng và dịch vụ như truy cập vào World Wide Web

Khi nói đến website và mạng máy tính thì cứ mặc định mạng đó là mạng Internet.

III. Nguyên lý hoạt động của một web

  1. Đầu tiên người dùng nhập vào trình duyệt một địa chỉ có dạng: https://abc.xyz, thực ra bạn chỉ cần gõ “abc.xyz” là trình duyệt sẽ tự hiểu và đổi thành đường dẫn ở trên.
  2. Trình duyệt gửi yêu cầu đến máy chủ DNS
  3. Hệ thống DNS trả kết quả phân tích tên miền trong đường dẫn đã gửi là abc.xyz, nó có địa chỉ máy chủ là một địa chỉ IP
  4. Sau khi nhận được địa chỉ IP – nơi lấy dữ liệu, trình duyệt sẽ tìm đến địa chỉ IP đã nhận – tức máy chủ chứa nội dung website.
  5. Máy chủ web nhận được yêu cầu truy xuất nội dung website và nó gửi một tập hợp các file bao gồm HTML, CSS , các tập tin đa phương tiện khác như âm thanh, hình ảnh (nếu có) cho trình duyệt
  6. Trình duyệt “dịch” các file mà máy chủ đã gửi thành trang web mà chúng ta nhìn thấy trên màn hình
